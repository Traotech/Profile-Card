# Profile Card

A simple and responsive profile card built with HTML, CSS, and JavaScript.  
It displays user information such as profile image, name, bio, social links, hobbies, and real-time data.

##  Features
- Profile image, name, and bio  
- Social media links (X, GitHub, LinkedIn)  
- Hobbies and dislikes section  
- Real-time dynamic content  
- Clean and responsive layout  

##  Tech Stack
- HTML  
- CSS  
- JavaScript  

##  Purpose
This project was built to practice frontend development and improve UI structuring using semantic HTML and modern styling.
